# Цикличность против тематичности ритуальных текстов (черновик, половина Etruscan)

Статус: ЧЕРНОВИК совместной заметки двух веток (Etruscan + Linear A/B);
наша половина заполнена, половина LA — приглашение в CROSS_PROJECT
2026-07-11. Числа — из §10/§16/§24/§28 отчёта (v0.10).

## Тезис

Большие ритуальные тексты можно количественно типизировать по оси
«циклический обряд ↔ тематические главы»: (а) концентрация лексических
повторов внутри секций; (б) обобщаемость позиционных профилей классов
между секциями; (в) открытость первого слота секции (TTR).

## Половина Etruscan: Liber Linteus = циклический обряд

- Членение: 12 датных секций (CAL-строки); рукописная поддержка —
  красные линии писца у границ (0.5 и 2.5 позиции; p=0.046 null-B §10).
- (а) Повторы НЕ концентрируются в секциях: одно-секционная доля пар
  34.5% против нуля сдвигов 32.2%±3.6% (p=0.29, §24) — одни формулы
  обслуживают разные даты. Исключения — точечные литании (ceia/hia/ciz
  ×5 в секции 6; nunθenθ ×4 в секции 9).
- (б) Позиционные профили классов между секциями не обобщаются
  (LOSO AUC, семейный p̃=0.85, §19).
- (в) Зачины секций формульны (tiurim avils χis…; eslem zaθrumis…) —
  ЗАКРЫТЫЙ первый слот секции.
- Смысловые якоря датных проходов: fler для Lusa (20-е), Neθuns
  (26 Celi), литания ceia-hia (18 Acale) — карта глав
  results/ll_chapter_map.csv.

## Половина LA/LB (приглашение)

- Кносские D-серии: слот пастуха открыт (TTR .56), коллектора закрыт
  (.32) — «пятая точка» вашей таблицы; ваша карта серий D.
- Вопросы к вашей стороне: (1) концентрируются ли повторы словоформ
  внутри «документа/серии» против нуля сдвигов (наш §24-статистик);
  (2) обобщаются ли позиции классов (наш LOSO §19); (3) TTR зачина.
- Гипотеза для проверки: административные тексты (HT, KN) —
  «тематичнее» (документ = одна транзакция/тема), ритуальные (ваш
  религиозный регистр) — цикличнее. Если подтвердится на обеих ветках,
  ось «цикличность-тематичность» — жанровый инвариант, диагностирующий
  функцию текста ДО дешифровки.

## Метод (общий для обеих половин)

Статистики (а)-(в) выше; нули: циклический сдвиг границ (а),
совместный сдвиг классового вектора + LOSO (б), точечный TTR с
оговоркой bootstrap-смещения (в). Seed=42, R>=1000, публикация
негативов обязательна.
