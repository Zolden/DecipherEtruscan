# Предрегистрация №4 (Etruscan): ретийский против ареального контроля

Дата заморозки правил: 2026-07-11. Ритуал прежний (№1 провал, №2 и №3
подтверждены): правила и скрипт коммитятся ДО прогона; прогон ОДИН;
публикация как вышло.

Мотив: §27 показал, что top-10 покрытие само по себе частично ареально
(LAT/FAL-контроль 73.3% при нуле 31%). Разведочный контраст «Raetic
93.1% против контроля 73.3%» требует собственного нуля — это и есть №4.

## Гипотеза

H1 (единственная): доля словоформ, оканчивающихся на один из top-10
MDL-суффиксов (results/mdl_suffixes_v1.csv, зафиксированы §9.4:
-s -al -sa -ial -i -a -e -as -ies -u), у ретийских словоформ TIR ВЫШЕ,
чем у латинско-фалискских словоформ корпуса (Бурман-разметка), сверх
случайного: нуль — перестановка языковых меток между объединёнными
словами двух срезов (R=10000, seed=42); статистика — разность долей.
Критерий: p <= 0.05.

## Замороженная процедура

- Слова: RAET — tir_words.csv language содержит 'Raetic', нормализация
  norm_any (§27), len>=4; LAT/FAL — записи корпуса v0.10 c lang∈{lat,fal},
  variant_of None, токены len>=4 без '-'; уникальные типы с обеих сторон.
- Покрытие: слово покрыто, если его конец (жадно 4..1) — суффикс top-10
  при остатке-основе >=3 (машинерия §19/§27).
- Нуль: объединить слова, переставить метки (размеры групп сохранены),
  разность долей; p = (1+#{null>=obs})/(R+1).
- Скрипт: tools/etr_prereg4_raetic.py; прогон один; лог как есть.
