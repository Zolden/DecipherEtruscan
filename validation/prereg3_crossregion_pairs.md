## Гипотеза

H1 (единственная): среди скелетных событий замещения (машинерия §13,
часть 2: имена → '*', реестр 48 операторов/6 классов, идентичный
скелетный контекст, разные памятники), ДОПОЛНИТЕЛЬНО ограниченных
парами записей с ИЗВЕСТНЫМИ И РАЗЛИЧНЫМИ регионами (поле region
Бурман-джойна), доля одноклассовых замещений выше нуля глобальной
перестановки операторов по инцидентностям (перестановка внутри полного
скелетного графа с регионами, события считаются по тем же правилам).
Критерий: p <= 0.05, R=1000, seed=42.

## Замороженная процедура

- Корпус v0.9 (sha 6ea6fe59…), канонический вид; записи 3–8 слов
  (токены len>=2, без '-') с непустым region.
- Реестр OPS и классы — БУКВАЛЬНО из tools/etr_minimal_pairs.py (§13).
- Событие: пара (запись A, запись B) в одной скелетной корзине
  (n, i, контекст), разные artifact_id, разные операторы, разные
  регионы.
- Нуль: R=1000; перестановка операторов по всем инцидентностям корзин
  (как §13), события пересчитываются с теми же фильтрами (включая
  регионы); p = (1+#{null>=obs})/(R+1).
- Скрипт: tools/etr_prereg3_crossregion.py; прогон один.
