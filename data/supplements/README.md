# Supplement-механизм

Замороженный `data/etr_corpus.pkl` НЕ редактируется. Любые добавления к
корпусу — только через CSV-файлы в этой папке: `tools/etr_freeze.py`
подхватывает их автоматически (в алфавитном порядке имён) и включает в
пересборку с `src = SUPP:<имя файла>`; после добавления заморозка
перегоняется, новый sha256 фиксируется в `data/etr_corpus.sha256`,
изменение коммитится вместе с логом.

## Схема CSV (кодировка UTF-8, заголовок обязателен)

| колонка      | обяз. | содержимое |
|--------------|-------|------------|
| `text`       | да    | транслитерация (конвенции ETP-части: θ χ φ ś σ, разметка Лейдена, `|` — граница строки) |
| `source_id`  | нет   | внешний ID (сигла ET/ETP/CIE, издание) |
| `city`       | нет   | город/регион |
| `date_from`  | нет   | датировка, год до н.э. (число, ранняя граница) |
| `date_to`    | нет   | датировка, год до н.э. (поздняя граница) |
| `translation`| нет   | перевод (английский, как в основном корпусе) |
| `language`   | нет   | `etr` (по умолчанию) / `lat` / `umb` / … |
| `provenance` | да*   | откуда взят текст: издание, страница, DOI/URL — БЕЗ этого поля supplement не принимается ревью, хотя загрузчик его не требует |
| `note`       | нет   | примечания |

## Кандидаты на первые supplement-ы (задокументированные пробелы корпуса)

1. **Скрижали Пирги** (ET Cr 4.4–4.5; золотые пластины с финикийской
   билингвой) — в корпусе ОТСУТСТВУЮТ (проверено: нет `tmia`,
   `θefarie`); источник: любое критическое издание (TLE² 874).
2. **Liber Linteus полностью** — в корпусе только 12 строк-цитат
   (LL 2–11); транслитерация не в открытом доступе (Woudhuizen 2013,
   van der Meer 2007) — этап 5 плана.
3. **Tabula Capuana полностью** — в корпусе только конкорданс-слова
   (CIEP:8682:*).
