Находки этого проекта, полезные сестринскому DecipherLinearA
(C:\OtherProjects\DecipherLinearA; github.com/Zolden/DecipherLinearA),
фиксируются здесь с датой и кратким описанием; о каждой — сказать

## 2026-07-08 — итоги §0–§1 DecipherEtruscan

1. **Позиционная типология операторов (структурная гипотеза для LA).**
   В этрусском начальную позицию монополизируют дейксис/прономиналы
   (mi «я» 89% нач, mini, указательные — все p̃_сем=0.0001), финальную —
   предикаты (lupu «умер» 67% фин, связка ame 55% фин); термины родства
   и титулы никогда не начальны. Проверяемая гипотеза для LA: финальные
   операторы (KU-RO «итого» и т.п.) — предикатного типа, начальные —
   дейктико-заголовочного; тест — на реестре §BC теми же нулями.
2. **Методика: Westfall–Young min-p** как точная реализация «семейного
   p_max» — векторизованный код в tools/etr_operators.py (симуляция
   позиций + ранги по семейству тестов); прямо переносим в LA-скрипты.
3. **Урок заморозки компилятивных корпусов.** В производных датасетах
   одна надпись живёт несколькими строками-чтениями (дипломатическое T /
   исправленное C — у нас это двоило статистику) и корпус может быть
   языково загрязнён (в «этрусском» датасете сидели умбрский, кельтский,
   фалискский, латынь). Решение: слияние чтений по построению + языковая
   чистка через кросс-таблицы номеров корпусов (у нас — Бурман, Zenodo
   10.5281/zenodo.17209666, CC0). Для LA: при интеграции внешних
   датасетов (SigLA и др.) проверять двойной счёт чтений и примеси
   (кипро-минойские? кносские дублеты?) тем же способом.
4. **Редкие графемы как маркёры одного памятника.** Символ ê у Larth
   оказался особым знаком ровно одного текста (Кортонская табличка,
   «перевёрнутый эпсилон» ɜ). Для LA: перед фонетическими выводами о
   редких знаках проверять их концентрацию по документам/писцам.
